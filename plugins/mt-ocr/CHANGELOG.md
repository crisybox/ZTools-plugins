# Changelog

## [1.0.0] - 2026-05-27

### 🎉 首次发布

馒头OCR — 多引擎 OCR 文字识别 ZTools 插件，支持截图/本地图片/剪贴板/拖放识别。

### ✨ 功能特性

- **多引擎支持**：腾讯云 OCR、百度云 OCR、阿里云 OCR，可任意切换
- **多种输入方式**：
  - 截图识别（框选屏幕区域）
  - 打开本地图片识别
  - 剪贴板粘贴识别
  - 拖放图片文件识别
- **结果操作**：
  - 一键复制识别文字到剪贴板
  - 保存识别结果为 txt 文件
  - 保存原始图片到本地
- **一键翻译**：识别结果可直接跳转到「易翻翻译」插件进行翻译
- **配置管理**：支持导入/导出 OCR 密钥配置

### 🔧 代码审查修复

- **百度云 Token 缓存校验密钥变更**：Token 缓存新增密钥一致性校验（`cachedApiKey`/`cachedSecretKey`），切换密钥时自动刷新 token，避免使用旧密钥的缓存导致认证失败
- **阿里云 OCR 排序传递性修复**：行重组改为两阶段稳定算法（先按 Y 排序归行，再每行按 X 排序），消除 `array.sort` 比较函数违反传递性导致排序不可预测的问题
- **异步竞态条件防护**：`handleRecognize` 增加图像快照比对，只在图片未发生变化时应用识别结果和状态，避免快速切换图片导致 UI 显示不一致
- **clearImage 状态重置**：清空图片时同步重置 `recognizing` 状态，避免识别中清图后加载动画残留
- **文件写入异常处理**：`handleSaveText`/`handleSaveImage` 增加 `try-catch`，磁盘空间不足或无写入权限时弹出友好错误提示

### 🛠 技术栈

- Vue 3 + Composition API
- Vite 6 + TypeScript
- ZTools 插件框架
