# Changelog

本文件记录 ZTools 开发百宝箱插件的版本变更历史。

## [2.3.3] - 2026-05-20

### 新增

- **压缩文本工具** — 将多行文本中的换行、制表符、多余空格压缩为单个空格
  - 支持移除代码注释（//、--、/* */）
  - 支持保留引号内空格（"..."、'...'、`...`）
  - 支持按指定宽度智能折行（在空格处断行，不会截断单词）
  - 显示压缩前后字符数、行数对比和节省百分比

## [2.3.2] - 2026-05-09

### 修复

- 时间戳识别正则精确匹配 10 位或 13 位，避免 11/12 位数字误匹配
- 第一项标签从「本地时间」改为「格式化时间」，避免切换时区后产生误导
- RFC 2822 格式动态使用所选时区偏移，不再硬编码 UTC
- 未来时间（60 秒内）显示「片刻后」而非「刚刚」
- 移除实时时钟 pulse 动画，提供更稳定的视觉体验

## [2.3.1] - 2026-05-09

### 变更

- 优化侧边栏分类：原 5 分类（随机生成、加密安全、文本、图像、日期时间）精简为 3 分类（随机生成、编码转换、开发工具）
- 简化插件描述和 README 工具列表，去除冗余功能说明

## [2.3.0] - 2026-05-09

### 新增

- **时间转换工具** — 显示当前实时时间，支持输入时间戳或日期字符串进行格式转换
  - 11 种时间格式：本地时间、带毫秒、星期、ISO 8601、UTC、RFC 2822、Unix 时间戳（秒/毫秒）、相对时间、路径格式、紧凑格式
  - 支持多时区切换（本地、UTC、北京、东京、纽约等 9 个常用时区）
  - 自动识别 Unix 时间戳（10 位秒 / 13 位毫秒）和 ISO 日期字符串
  - 点击任意格式行一键复制

### 变更

- 插件描述新增「时间转换」
- 项目结构文档同步更新

## [2.2.0] - 2026-04-27

### 新增

- **HTML 预览工具** — 粘贴或输入 HTML 代码，实时预览渲染效果
  - 实时预览（300ms 防抖）
  - 从剪贴板一键粘贴
  - 导出为 HTML 文件

### 变更

- 添加插件分类字段（`categories`）
- 更新作者名称为「咖啡八杯」

## [2.1.0] - 2026-04-20

### 新增

- **二维码工具** — 生成二维码或将二维码图片解码为文字
  - 支持文件上传、截图和粘贴
- **中文转拼音工具** — 将中文转换为拼音命名格式
  - 全拼、驼峰、下划线、常量、首字母等多种格式

### 变更

- 插件名称从 `myplugins` 更改为 `devbox`，更换 logo
- 使用 `crypto.getRandomValues()` 替代 `Math.random()` 提升随机数安全性
- 身份证号按照国标格式（GB 11643）生成，包含校验位

### 修复

- 修复身份证号校验位计算错误的问题
- 修复属性命名不规范、中文名简化等代码审查问题

## [2.1.0-beta] - 2026-04-17

### 新增

- 项目初始版本
- **随机身份生成器** — 虚拟身份信息（姓名、身份证号、手机号、地址等 16 个字段）
- **随机密码生成器** — 可配置长度、字符集，实时显示密码强度
- **随机数字生成器** — 范围随机数，支持批量和自定义小数位数
- **UUID 生成器** — 符合 RFC 4122 v4 规范
- **随机颜色生成器** — HSL 滑块调色，支持 HEX/RGB/HSL 多格式复制
- **加密签名生成器** — 支持 MD5、SHA-1、SHA-256、SHA-512、HMAC-SHA256、HMAC-SHA512
