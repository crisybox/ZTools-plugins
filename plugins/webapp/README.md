# ZTools WebApp 插件

> 添加你的 Web 应用，一键直接加载。

## ✨ 功能

- **添加 Web 应用** — 输入名称和地址，支持自定义图标
- **Basic Auth 认证** — 支持需要用户名密码认证的网页，自动登录
- **默认加载** — 设置某个应用为默认，打开插件时自动打开
- **应用管理** — 右键图标可编辑，支持删除应用
- **导入导出** — 一键备份配置，方便在其他设备恢复
- **暗色/亮色主题** — 自动跟随系统主题切换

## 📖 使用说明

### 添加应用

1. 点击左侧栏顶部的 **+** 按钮
2. 填写应用名称和网址（如 `https://github.com`）
3. 点击「自动获取」可自动抓取网站图标
4. 如需认证，勾选「启用 Basic Auth 认证」，填入用户名和密码
5. 勾选「默认加载」可设为打开插件时自动加载的应用
6. 点击「添加」完成

### 编辑/删除应用

- **右键点击** 侧边栏的应用图标 → 打开编辑对话框
- 在编辑对话框中可修改所有配置，或点击「删除」移除应用

### 导入导出配置

1. 点击左下角 **⚙️ 设置** 按钮
2. **导出** — 将当前所有应用配置导出为 JSON 文件
3. **导入** — 选择 JSON 配置文件，支持合并或覆盖模式

### Basic Auth 认证

适用于需要输入用户名和密码才能访问的网页（如内部管理系统、私有面板等）：

1. 添加应用时勾选「启用 Basic Auth 认证」
2. 输入用户名和密码
3. 保存后即可正常访问，无需每次手动输入

## 🚀 安装

将 `dist/` 目录中的文件复制到 ZTools 插件目录：

```
%APPDATA%\ZTools\plugins\WEBAPP\
```

重启 ZTools 即可使用。

## 🔧 开发

```bash
npm install     # 安装依赖
npm run dev     # 开发模式（热更新）
npm run build   # 构建生产版本
```

开发服务器启动后，ZTools 会自动加载 `http://localhost:5173` 的开发版本。
